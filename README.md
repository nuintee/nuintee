## ■ 私について

フロントエンドを主軸にしつつ、DB 設計 / REST API 設計・実装まで横断する Web エンジニアです。
0→1 の新規開発から大規模リプレイス、長期の保守運用まで一通り経験し、2025 年には初の PM として 9 名規模のプロジェクトを牽引しました。
**領域や役割を限定しすぎず、課題を拾って完遂まで導く**のが強みです。

- 主力: **TypeScript / React / Next.js**
- 得意: 開発基盤構築（Storybook / MSW など）、FE/BE を横断した0⇨1開発
- 業務領域: 金融 / 人材・求人 / ヘルスケア（健康経営）
